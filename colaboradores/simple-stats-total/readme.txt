Simple summary statistics on visited pages.


== Description ==

The Simple Stats Total plugin allows you to collect and view page visits statistics.
The stats is display most visited pages, referer URLs, IP addresses, web browsers and versions, visitor operating systems.
The stats is summarized and sorted in descending order.


== Installation ==

#### Installing the plugin
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Simple Stats Total** and click "*Install now*"
1. Alternatively, download the plugin and upload the contents of `simple-stats-total.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
1. Activate the plugin


== Frequently Asked Questions ==

= How to access statistics page? =

Statistics page can be accessed in your WordPress admin panel by link in Tools menu *Tools > Simple Stats Total*.

= How to clear statistics data? =

Use "Purge All Stats" button for this purpose.

= I've activated the plugin but nothing happens. =

Please check if your server is running PHP version 5.3 or later. The plugin will not do anything if you're on an older version of PHP.
